JAVA GUI Based Program

Java was taught extensively and repeatedly during my time at University. This first project is a GUI based program that simulated a warehouse system with the following functionalities;

    • Load in a list of Customers from a file
    • Load in a list of Orders from a file
    • Generate and populate an activity log
    • Create & remove workers
    • Set the workers status to active or inactive
    • Process orders with each worker

In addition to the gui based program, there was also in depth testing of all classes, this can be seen in greater detail in the attached report. If you select the download option, the zip file also includes the entirety of the program.



JAVA distributed program

With this project we were provided with some very basic broken code that was intended to count the frequency and occurrences of each letter of the alphabet, for a given hard-coded text document. After first having to make the program work for a single client, I then had to expanding its scope to include the following features;

    • Fix the existing bugs in the code.
    • Add functionality to specify and select multiple different text files.
    • Add the functionality to only count vowels, consonants or both.
    • Add the functionality to count occurrences of a user specified word.
    • Add the functionality to count occurrences a word with a length greater than that which is declared by the user.
    • Add the functionality to increase the number of clients, maintaining asynchronous opperation.

The first paper here represents a large amount of code all relating to the core concepts of distributed programming.
