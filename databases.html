Big Data

This report was made in the final year of my undergraduate degree as part of the Big Data module.
For this assignment I was required to source a large dataset and conduct data cleansing and analysis. The dataset chosen was based around HGV test pass rates. The report covers the following topics;

    • Introduction to Big Data
    • Introduction to the dataset
    • Methodologies for importing the data
    • Data cleansing processes and documentation
    • Visualisation of the data
