Embedded Systems Using Hardware


Throughout my studies I was fortunate to work with physical hardware such as Adrunio's on multiple occasions. However, this most recent project was initially built virtually using the TinkerCad Software before being built using actual hardware.


Multi Device Home Automation System

For this particular project I was tasked with designing and creating, in physical hardware, a multi-functional remote based home automation system. The brief required that five separate devices/systems be created meeting the following specifications;

    • Device 1 is a remote-control device. This device is to be used to control all other devices, whilst also showing the individual system status’s.
    • Device 2 is an automated external light, that functions on a user’s pre set light level, whilst also having a manual on/off override.
    • Device 3 is a control for a servo to open and close an external gate.
    • Device 4 controls the interior light. The room has a simple (on/off) but also a dimmer function, meaning that the light level can be altered.D
    • Device 5 passive fire detection (infra-red).
