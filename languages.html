Programming languages

Below are some examples of the different programming languages that I have had experiance with.
They represent a wide range of approaches from core fundamentals to OOP and multi-threaded operations.


JAVA

Java was taught extensively and repeatedly during my time at University.This first project is a GUI based program that simulated a warehouse system with the following functionalities;

    • Load in a list of Customers from a file
    • Load in a list of Orders from a file
    • Generate and populate an activity log
    • Create & remove workers
    • Set the workers status to active or inactive
    • Process orders with each worker

In addition to the gui based program, there was also in depth testing of all classes, this can be seen in greater detail in the attached report. If you select the download option, the zip file also includes the entirety of the program.
